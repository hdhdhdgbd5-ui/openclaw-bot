## System Architecture

### Files Created

1. **class_action_tracker.md** - Settlement database with deadlines
2. **scripts/class_action_hunter.py** - Main claim tracking system

### How The System Works

1. **Settlement Discovery**: Scans classaction.org and topclassactions.com
2. **Deadline Tracking**: Monitors upcoming deadlines (next 30 days highlighted)
3. **Eligibility Checking**: Matches user profile against settlement criteria
4. **Claim Filing**: Browser automation to submit claim forms
5. **Status Tracking**: JSON-based tracker for filed claims

### Claim Filing Process (For Each Settlement)

1. Navigate to settlement website
2. Find "Submit Claim" form
3. Fill required fields:
   - Name
   - Email
   - Address (if required)
   - Purchase details (if applicable)
4. Submit form
5. Record confirmation number
