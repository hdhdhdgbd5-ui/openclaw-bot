# Alternative Marketing Channels

## Platforms That DON'T Require Authentication

### 1. ProductHunt (Requires auth but easy to create)
- Need: Email only
- Can create account with temp email

### 2. Hacker News (No auth for reading, karma needed to post)
- https://news.ycombinator.com
- Submit via: https://news.ycombinator.com/submit

### 3. Reddit (Some subreddits allow link posts without account)
- r/bookmarks
- r/webmarketing
- r/assistance

### 4. Indie Hackers (Requires auth)
- https://www.indiehackers.com

### 5. Dev.to (Free, easy signup)
- https://dev.to
- API available for auto-posting

### 6. Medium (Requires auth but easy)
- https://medium.com

### 7. GitHub Explore
- https://github.com/explore
