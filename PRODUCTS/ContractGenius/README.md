# ContractGenius

## AI-Powered Contract Analysis Platform
Production-ready, enterprise-grade contract analyzer.

### Features
- ✅ PDF/DOC/TXT Upload
- ✅ AI Contract Analysis (risk scoring)
- ✅ Key Term Extraction
- ✅ Red Flag Detection
- ✅ Plain English Summaries
- ✅ Export Annotated PDF
- ✅ Premium $9.99 tier

### Quick Start
```bash
cd backend
pip install -r requirements.txt
python main.py

# Frontend
cd frontend
npm install
npm run dev
```

### API Keys Needed
- OpenAI API Key (sk-...) for AI analysis
- Optional: Anthropic Claude for complex legal documents

### Deployment
Backend: Render.com (free)
Frontend: GitHub Pages (free)
